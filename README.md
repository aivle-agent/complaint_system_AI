# complaint_system_AI

지역사회의 실질적인 문제를 해결할 수 있는 시스템의 효율성을 높인다. ( 지역 문제 표현력 향상 + 해결 과정 행정 과부화 감소)

활용방안

일반인 기존 사례를 통한 민원에 대한 처리가 어떻게 되었는지 볼 수 있다.
공무원 기준으로는 담당하지 않는 영역의 민원이 줄어들어 업무부담이 줄어든다.

기대 효과

질문교정 → 민원을 제기할 부서를 찾음으로써 해결가능성을 높인다.
부서 추천 → 민원을 제기할 부서를 찾음
예상 답변 → 문제 해결에 대한 예상을 해볼 수 있다. + 민원을 좀더 적용가능성 높게 다듬을 수 있다.

국민청원시스템 → 좋아요 수가 적으면 묻힌다.

국민신문고 - 공개제안 관심도가 낮아도 의견이 반영될 수 있다.

순서:
민원 인풋 -> 질문 교정 및 검증 -> 답변 전략 설정 -> 답변 생성 -> 최종 답변
